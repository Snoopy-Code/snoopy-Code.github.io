Docker创建

安装Docker先yum install epel-release

然后yum install -y docker-io会出现

这里是因为这句话用源下载，但是源在国外，被墙了。所以使用不了

这里可以使用yum install https://get.docker.com/rpm/1.7.1/centos-6/RPMS/x86_64/docker-engine-1.7.1-1.el6.x86_64.rpm

安装完之后我又出现了了Get http:///var/run/docker.sock/v1.19/containers/json: dial unix /var/run/docker.sock: no such file or directory. Are you trying to connect to a TLS-enabled daemon without TLS?这个错误，很简单，我没有开启docker服务=。=

开启后，使用docker version查看就可以了
