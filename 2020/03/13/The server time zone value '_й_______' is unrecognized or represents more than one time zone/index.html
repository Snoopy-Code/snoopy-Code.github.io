The server time zone value '_й_______' is unrecognized or represents more than one time zone

错误原因：

使用了 Mysql 最新版驱动

解决方案：

在项目代码-数据库连接 URL 后，加上 （注意大小写必须一致）
?serverTimezone=UTC
